Data Release

The following data products are released with the paper:

File Description
morpheus_GOODS_spheroid.fits The pixel-level classifications for the spheroid class
morpheus_GOODS_disk.fits The pixel-level classifications for the disk class
morpheus_GOODS_irregular.fits The pixel-level classifications for the irregular class
morpheus_GOODS_ps_compact.fits The pixel-level classifications for the point source/compact class
morpheus_GOODS_background.fits The pixel-level classifications for the background class
morpheus_GOODS_3dhst_catalog.fits Morphological classifications for objects in teh 3DHST Survey
morpheus_GOODS_all.fits All of the above files compressed together
